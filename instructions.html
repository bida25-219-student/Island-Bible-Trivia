Game Instructions

  • Start the game from the Home page by selecting Start Game.
  • Choose a mode: Solo, Double Team, Triple Team, Quad.
  • Enter valid player names for the selected mode.
  • Select a difficulty level: Easy, Normal, Hard, Very Hard, Legendary.
  • Answer Bible trivia questions. Each question has a 10-second timer.
  • Correct answers increase your score; wrong answers show the correct answer in green.
  • The game continues for all 40 questions of the selected difficulty.
  • At the end, each player’s total score is displayed. The highest score wins.
  • Use your Bible knowledge to answer as many questions correctly as possible!
